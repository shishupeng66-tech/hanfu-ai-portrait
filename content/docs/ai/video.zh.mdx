## 概述

视频生成是**异步**流程。与对话和图像生成不同，结果不会立即可用。

## 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/video/generate` | POST | 提交生成请求 |
| `/api/video/status` | GET | 查询任务状态 |

## 异步工作流

1. **提交** — 用户发送提示词到 `/api/video/generate`
2. **任务创建** — 火山引擎返回 `taskId`
3. **轮询** — 客户端轮询 `/api/video/status?taskId=xxx`
4. **完成** — 状态为 `completed` 时，`resultUrl` 包含视频

## 请求

```json
{
  "prompt": "一只猫在弹钢琴",
  "referenceImage": "可选的base64或URL"
}
```

支持文生视频和图生视频两种模式。

## 积分消耗

- **50 积分**每次视频生成
- 积分在提交时扣除，任务失败时退还

## 任务状态

| 状态 | 说明 |
|------|------|
| `pending` | 已提交，等待开始 |
| `processing` | 生成中 |
| `completed` | 视频就绪，可通过 `resultUrl` 获取 |
| `failed` | 生成失败，积分已退还 |
