## 什么是 Sistine Starter？

Sistine Starter 是一个**生产级 AI SaaS 模板**，专为快速启动商业化 AI 产品而设计。包含完整的用户认证、订阅计费、积分系统、AI 功能集成（对话/图像/视频生成）、管理后台和国际化支持。

### 核心特性

- **用户认证** — 邮箱密码 + Google OAuth（Better Auth）
- **积分计费** — 按次计费的积分系统 + 可审计的积分账本 + 订阅计划（Creem 支付）
- **AI 集成** — 对话、图像生成、视频生成（火山引擎 / 豆包 API）
- **管理后台** — 用户管理、积分调整、订阅控制
- **国际化** — 完整的中英文支持（next-intl）
- **邮件系统** — 交易邮件（Resend）：验证、密码重置、购买确认
- **营销页面** — 首页、定价、博客（MDX）、法律页面

### 技术栈

| 层级 | 技术 |
|------|------|
| 框架 | Next.js 16 (App Router) + React 19 |
| 样式 | Tailwind CSS + Framer Motion |
| 数据库 | PostgreSQL + Drizzle ORM |
| 认证 | Better Auth |
| 支付 | Creem |
| AI | 火山引擎（豆包 API） |
| 邮件 | Resend |
| 国际化 | next-intl |
